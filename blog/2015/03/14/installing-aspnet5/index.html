Installing ASP.NET 5 (beta 3)

Installing VS 2015 and the ASP.NET 5 CLI tools

The easiest, but slowest, way to install ASP.NET 5 is through Visual Studio 2015.

Alternatively, you can install the command line tools by following the instructions below. These are instructions are for the beta 3 versions of the tools as posted on ASP.NET 5 on GitHub. Beginning with beta 4 these will be obsolete.

Visual Studio 2015

Go here download a free copy of Visual Studio 2015 CTP.

Download Visual Studio 2015 CTP 6

Command Line Tools

These instructions are found in more complete form on the ASP.NET 5 Home GitHub repository.

Windows

Execute the following in Command Prompt (cmd.exe)

@powershell -NoProfile -ExecutionPolicy unrestricted -Command "iex ((new-object net.webclient).DownloadString('https://raw.githubusercontent.com/aspnet/Home/master/kvminstall.ps1'))"

OS X

You must have >= Mono 3.6.0 installed.

If you have brew, the following will the correct version of Mono and the KVM. Execute these in the Terminal.

brew tap aspnet/k
brew install kvm
source kvm.sh

Linux

You must have Mono 3.4.1 or greater installed. Execute these on the command line.

curl -sSL https://raw.githubusercontent.com/aspnet/Home/master/kvminstall.sh | sh && source ~/.k/kvm/kvm.sh

Beta 4
